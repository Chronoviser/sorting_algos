# Sorting Algortihms

This app Implements, Visualizes and gives details about various sorting algorithms.
### Sorting Algorithms included in this Application are:
1. Bubble Sort
2. Insertion Sort
3. Selection Sort
4. Quick Sort
5. Merge Sort
6. Heap Sort
7. Bucket Sort
8. Radix Sort
